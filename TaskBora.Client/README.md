# TaskBora Frontend (React + Supabase)

This folder hosts the original React application generated by Lovable, now nested under `frontend/` so it sits alongside the .NET backend.

## Prerequisites
- Node.js 18+
- Supabase project credentials exposed via environment variables (see `.env.example` guidance below)

## Setup
1. `cd frontend`
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`

Create a `.env` file in `frontend/` with your Supabase settings:
```
VITE_SUPABASE_URL=<your-url>
VITE_SUPABASE_PUBLISHABLE_KEY=<your-publishable-key>
```

The Supabase client lives in `src/integrations/supabase/client.ts` and expects these values.
