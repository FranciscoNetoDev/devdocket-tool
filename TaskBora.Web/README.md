# TaskBora Frontend (React targeting .NET API)

This folder hosts the original React application generated by Lovable. It now sits at the solution root beside the .NET backend projects and calls the new API by default.

## Prerequisites
- Node.js 18+
- API base URL exposed via `VITE_API_BASE_URL` (defaults to `http://localhost:8080` when using Docker Compose)
- Supabase credentials only if you still rely on the legacy auth flow (to be replaced by the .NET backend)

## Setup
1. `cd TaskBora.Web`
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`

Create a `.env` file in `TaskBora.Web/` with your API settings (and optional Supabase auth keys during the transition):
```
VITE_API_BASE_URL=http://localhost:8080
VITE_SUPABASE_URL=<optional-for-auth>
VITE_SUPABASE_PUBLISHABLE_KEY=<optional-for-auth>
```

Task and project data now flow exclusively through the .NET Web API. Supabase is only used for the remaining authentication pieces until the new identity flow ships.
